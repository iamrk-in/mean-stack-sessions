-> What is jQuery?
    - jQuery is a light-weight javascript library
    - 'write less, do more'
    - It's easy-to-use API's makes things like HTML document travesel and manipulations, event handlers, adding animation effects to a web page
    - to create an Ajax based applications

-> What you can do with jQuery?
    - can easily select elements to perform manipulation
    - create effects like show or hide elements, sliding transition etc., 
    - complex css animation
    - manipulation DOM elements and their attributes 

-> Advantages: 
    - saves a lot of time
    - simplify common javascript tasks
    - easy to use
